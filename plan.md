# 작업 계획: 모니터 테두리 순회 이동 기능 추가

## 목표
현재 작업 표시줄(하단)에서만 이동하는 펫을 모니터의 4개 테두리(하단 -> 우측 -> 상단 -> 좌측)를 따라 순회하도록 기능을 확장합니다.

## 작업 범위
- [ ] **백엔드 (Rust)**: `lib.rs` 이동 로직 수정
    - 4단계 상태(BOTTOM, RIGHT, TOP, LEFT) 정의 및 전환 로직 추가
    - 상태별 좌표(X, Y) 업데이트 로직 구현
    - 방향 상태를 프론트엔드로 전송하는 이벤트 추가
- [ ] **프런트엔드 (React)**: `MainWindow.tsx` 수정
    - 이동 방향에 따른 스프라이트 회전(`transform: rotate`) 적용
    - 말풍선 위치 보정
