# 설정 기능 확장: 폰트/언어 설정

## 목표
설정 화면의 "설정" 탭에 폰트 크기, 폰트 변경, 언어 변경 기능 추가.

## 범위
- 설정 탭에 폰트 크기 콤보박스 (최대 20), 폰트 선택, 언어 선택 추가
- 한글/영문 번역 리소스 파일 생성 (`src/locales.ts`)
- 모든 UI 텍스트를 번역 함수 `t()` 로 교체
- 언어: "시스템 언어" (기본), "한글", "영어" 콤보박스
- 시스템 언어가 한/영이 아니면 영문 표시
- 폰트/언어 설정은 localStorage에 저장, Tauri 이벤트로 윈도우 간 동기화

## 승인 완료
- 모든 UI 텍스트를 번역 키로 교체 (기존 동작 변경)
- `src/locales.ts` 신규 파일 추가

## 작업 단계

### 1단계: 번역 리소스 생성
- [ ] `src/locales.ts`: 한글/영문 번역 객체 + `t()` 함수 + 언어 감지 로직

### 2단계: App.tsx 수정
- [ ] 폰트 크기/폰트/언어 상태 추가 + localStorage 영속화
- [ ] 설정 탭에 폰트 크기 콤보박스, 폰트 선택, 언어 콤보박스 UI 추가
- [ ] 모든 하드코딩 텍스트를 `t()` 로 교체
- [ ] 폰트 설정을 메인 윈도우 말풍선에 적용
- [ ] Tauri 이벤트로 설정 동기화

### 3단계: 검증
- [ ] `npx tsc --noEmit` 통과
- [ ] `cargo build` 통과 (Rust 변경 시)
