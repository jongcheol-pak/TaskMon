# Releases

## v0.1.0 (2026-04-28)

### 새로운 기능
- 일본어, 중국어(간체·번체) UI를 지원합니다. 설정 → 일반에서 언어를 직접 선택하거나 시스템 언어에 따라 자동 선택할 수 있습니다.
- 모니터링 항목에 GPU 사용률이 추가되었습니다. Windows 작업 관리자와 동일한 방식으로 측정하며, 모니터링 메시지 조건에서도 GPU를 사용할 수 있습니다.

### UI/UX 개선
- 정보 탭에서 업데이트 확인이 실패할 때(네트워크 오류 등) "업데이트 확인에 실패했습니다." 대신 "최신 버전입니다."로 표시하여 불필요한 오류 안내를 줄였습니다.
- 트레이 메뉴의 "중지 → 시작"이 앱을 종료 후 재실행한 것과 동일한 초기 상태로 복구되도록 동작합니다. 캐릭터가 비정상 위치/모드에 빠졌거나 표시가 어긋난 경우, 앱을 종료하지 않고 트레이 토글만으로 정상 상태로 되돌릴 수 있습니다 (사용자 설정은 유지됩니다).

### 보안
- 자동 업데이트로 받은 설치 파일의 SHA256 무결성을 자동으로 검증합니다. 검증에 실패하면 받은 파일을 자동으로 삭제하고 오류를 안내합니다.
- 릴리즈 노트에 체크섬이 없는 경우, 다운로드를 진행할지 한 번 더 확인하는 안내가 표시됩니다.

### 정리
- "프로그램 추가/제거" 항목과 설치 마법사에 게시자/저작권/카테고리 정보가 표시되도록 보강했습니다.
- 설치 안내 문서에 Windows SmartScreen 경고 대응 방법과 SHA256 체크섬으로 무결성을 직접 확인하는 방법을 추가했습니다.
- 설치 마법사가 시스템 언어에 맞춰 한국어/영어/일본어/중국어(간체·번체)로 표시됩니다. 지원 언어가 아니면 영어로 표시됩니다.
