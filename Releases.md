# Releases

## v0.2.0 (2026-05-06)

### UI/UX 개선
- 펫이 벽을 등반·하강하거나 다른 모니터로 이동할 때 잠깐 멈췄다가 다시 움직이던 동작을 제거하여, 회전과 이동이 끊김 없이 자연스럽게 이어지도록 개선했습니다.
- 펫이 등반·하강·다른 모니터로 이동·좌우 반전할 때 회전 보간 중간에 비스듬히 기울어진 모습으로 잠깐 보이던 글리치를 제거하여, 방향 전환이 즉시 반영되도록 변경했습니다.
- 펫이 모니터 상단을 따라 이동할 때 메일 알림 말풍선이 펫과 겹쳐 보이던 문제를 해결했습니다.

### 버그 수정
- 메일 알림이 설정한 표시 시간이 지나도 사라지지 않고 계속 표시되는 문제를 수정했습니다.
- 메일 제목 끝부분에 깨진 글자(`�`)가 표시되던 경우, 깨진 부분만 제거하여 읽을 수 있는 부분이 깔끔하게 표시되도록 변경했습니다.

## v0.1.0 (2026-04-29)

### 새로운 기능
- POP3 메일 알림 기능이 추가되었습니다. 설정 화면의 새 "메일 알림" 탭에서 POP3 서버를 등록하면 1~60분 간격으로 신규 메일을 확인하고, 새 메일이 도착하면 펫 머리 위에 "📧 메일 도착" 헤더와 함께 받은 시간 / 보낸 사람 / 제목을 말풍선으로 표시합니다. 표시 시간은 1~600초로 설정할 수 있고 모든 알림 중 가장 우선 표시됩니다. 신규 메일이 2건 이상이면 말풍선 위쪽에 "새메일+N" 배지가 함께 표시됩니다.
- 메일 알림 표시 중 펫을 좌클릭하면 시간과 무관하게 즉시 닫히고 다음 메일이 있으면 이어서 표시됩니다.
- 메일 도착 시점이 90일 이상 지난 오래된 메일은 알림으로 표시되지 않습니다. 메일 식별자 기록도 90일 기준으로 자동 정리되어 데이터가 무한정 누적되지 않습니다.
- 일본어, 중국어(간체·번체) UI를 지원합니다. 설정 → 일반에서 언어를 직접 선택하거나 시스템 언어에 따라 자동 선택할 수 있습니다.
- 모니터링 항목에 GPU 사용률이 추가되었습니다. Windows 작업 관리자와 동일한 방식으로 측정하며, 모니터링 메시지 조건에서도 GPU를 사용할 수 있습니다.

### UI/UX 개선
- 정보 탭에서 업데이트 확인이 실패할 때(네트워크 오류 등) "업데이트 확인에 실패했습니다." 대신 "최신 버전입니다."로 표시하여 불필요한 오류 안내를 줄였습니다.
- 트레이 메뉴의 "중지 → 시작"이 앱을 종료 후 재실행한 것과 동일한 초기 상태로 복구되도록 동작합니다. 캐릭터가 비정상 위치/모드에 빠졌거나 표시가 어긋난 경우, 앱을 종료하지 않고 트레이 토글만으로 정상 상태로 되돌릴 수 있습니다 (사용자 설정은 유지됩니다).
- 메일 알림 설정에서 다른 계정으로 변경 후 저장하면 이전 계정 데이터가 자동으로 정리되어, 새 계정의 모든 메일이 한꺼번에 알림으로 쏟아지지 않습니다.
- 메일 알림 설정에 "처음 켤 때는 기존 메일이 모두 '읽은 상태'로 등록된다"는 안내가 추가되어 첫 사용 시 동작을 명확히 보여줍니다.
- 메일 알림 "테스트" 버튼 결과가 별도 영역에 표시되며 3초 후 자동으로 사라집니다. 폴링 상태 메시지가 테스트 결과를 덮어쓰는 문제가 해결되었습니다.

### 보안
- 메일 알림 비밀번호는 Windows 자격 증명 보호(DPAPI)로 암호화되어 현재 사용자 계정 키에 묶여 저장됩니다. 다른 사용자나 다른 PC에서는 복호화되지 않으며, 평문 비밀번호는 메모리에서도 송신 직후 즉시 지워집니다.
- 메일 알림 시 아이디/비밀번호가 잘못된 경우 폴링이 자동으로 정지되어 잠금 위험을 줄입니다. 사용자가 자격 증명을 다시 저장하면 폴링이 재개됩니다.
- 자동 업데이트로 받은 설치 파일의 SHA256 무결성을 자동으로 검증합니다. 검증에 실패하면 받은 파일을 자동으로 삭제하고 오류를 안내합니다.
- 릴리즈 노트에 체크섬이 없는 경우, 다운로드를 진행할지 한 번 더 확인하는 안내가 표시됩니다.

### 정리
- "프로그램 추가/제거" 항목과 설치 마법사에 게시자/저작권/카테고리 정보가 표시되도록 보강했습니다.
- 설치 안내 문서에 Windows SmartScreen 경고 대응 방법과 SHA256 체크섬으로 무결성을 직접 확인하는 방법을 추가했습니다.
- 설치 마법사가 시스템 언어에 맞춰 한국어/영어/일본어/중국어(간체·번체)로 표시됩니다. 지원 언어가 아니면 영어로 표시됩니다.
